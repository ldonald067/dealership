Walk through the full customer experience step-by-step, narrating what a first-time user would see and feel.

Steps:
1. Ensure dev server is running
2. Start at the landing page (http://localhost:3000)
3. Walk through each step a customer would take:
   a. Landing page — first impression
   b. Registration — how easy/inviting is signup?
   c. Dashboard — does the user know what to do next?
   d. Vehicle browsing — is it fun to explore? Do filters help?
   e. Vehicle detail + lease calculator — is the financial info clear and trustworthy?
   f. Document upload — is it obvious what's needed? Does it feel secure?
   g. Credit application — is it intimidating or approachable?
   h. Appointment booking — is it smooth and clear?
   i. Status tracking — does it reduce anxiety about the process?
4. At each step, screenshot the page and narrate:
   - What the user sees
   - What they're likely feeling (confused? confident? delighted?)
   - What might make them abandon the process
   - What's working well
5. Give an overall score 1-10 and top 3 improvements that would have the biggest impact on conversion/completion
