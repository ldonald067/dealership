- Page name/route

3. Create the new page file at the correct path under `src/app/(customer)/` or `src/app/(dealer)/`
4. Use the project's established component patterns, Tailwind theme, and shadcn/ui components
5. If API endpoints are needed, scaffold those too under `src/app/api/`
